# curse-clause-header
Inserts the following comment header to every file you commit to:

```js
/*
CURSE:

To all future maintainers of this source code:

Woe unto those who dare to modify my contributions herein!  A curse no shorter than six months shall befall the modifying author and their family!  In the likely event of the modifying author's demise prior to six months, this curse shall propagate to their manager for the remaining duration.

Modifications included under this curse:
- changes to the source code
- changes to this curse clause
- changes to the directory in which this file is stored
- any attempt to dilute this curse by providing spurrious source code surrounding the cursed source code
- changes in usage of the exported APIs in this cursed code in an attempt to mitigate the impact of this curse
*/
```
